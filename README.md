# Python for Data Science

A simple repository for learning Python for Data Science.

## Getting Started

This repository is designed to be simple and beginner-friendly. You don't need to set up virtual environments or complex tooling - just clone and start learning!

## Requirements

- Python 3.x installed on your system
- Basic Python packages (install as needed): `pip install numpy pandas matplotlib scikit-learn jupyter`

## Usage

Simply clone this repository and start working with Python files or Jupyter notebooks directly:

```bash
git clone https://github.com/jaytakahashii/Python-for-Data-Science.git
cd Python-for-Data-Science
python your_script.py
```

Or launch Jupyter:

```bash
jupyter notebook
```
